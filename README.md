# Nutritionist Website

Portfolio website for nutritionist Альмира Орлова with individual consultations, school support programs, and recipes.

## Tech Stack

- React 18 + React Router
- Vite
- Responsive design (mobile-first)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Deploy the `dist/` folder to any static host (Vercel, Netlify, etc).
